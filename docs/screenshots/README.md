# Screenshots de Invocação

Esta pasta contém screenshots das invocações manuais do Lambda para documentação.

## Arquivos

- `console-invocation.png` - Screenshot da invocação via Console AWS (Test tab)
- `cloudwatch-logs.png` - Screenshot dos logs no CloudWatch

## Como Adicionar Screenshots

Após realizar as invocações manuais:

1. Capturar tela do Console AWS mostrando:
   - Payload de entrada
   - Resposta JSON
   - Logs de execução
   
2. Capturar tela do CloudWatch mostrando:
   - Log stream recente
   - Mensagem "Hello World invoked"
   - Métricas de execução (duration, memory)

3. Salvar os arquivos nesta pasta com nomes descritivos
